Среди наиболее значимых проектов можно выделить следующие:

  • Представление интересов заказчика по строительству 5-ти-звездочной гостиницы в споре с подрядчиком о взыскании сумм неотработанных авансов, стоимости неиспользованных материалов, убытков;
  • Представление интересов собственника недвижимого имущества в споре о виндикации имущества, связанного с объектом недвижимого имущества;
  • Представления интересов иностранных поставщиков товаров по спорам с белорусскими покупателями о взыскании сумм основного долга, неустойки, процентов;
  • Представление интересов клиента при реализации проекта по строительству многофункционального торгово-развлекательного комплекса с гостиницей с общим объемом инвестиций около 150 млн. долларов США;
  • Представление интересов клиента при реализации проект по строительству пятизвездочного отеля в г. Минске;
  • Консультации в ходе заключения инвестиционного договора по строительству горно-обогатительного комплекса по производству калийных удобрений в Республики Беларусь с общим объемом инвестиций около 1,5 млрд. долларов США;
  • Участие в проектах по юридической оценке белорусских компаний (сфера телекоммуникаций, розничная торговля, торговля нефтепродуктами, банки).
